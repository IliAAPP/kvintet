README для бэк
Описание проекта
Этот проект представляет собой комбинацию клиентской и серверной частей для мобильного приложения. Клиентская часть разработана с использованием React Native, а серверная часть написана на Go. Приложение предоставляет пользователю функционал, связанный с личным кабинетом, покупкой и арендой, сканированием документов, доступом к личным документам и ходом работы.
Структура проекта
Проект состоит из двух основных частей: клиентской и серверной.
Клиентская часть
Клиентская часть приложения разработана с использованием React Native. В основном файле MainPage.js определены компоненты кнопок, таких как PersonalProfileButton, BuyAndRentButton, ScanDocumentsButton, MyDocumentsButton и WorkflowButton. Каждая кнопка имеет свою собственную логику обработки нажатия, основанную на навигации между различными экранами.
Серверная часть
Серверная часть приложения написана на языке Go. Основной файл main.go определяет обработчики для различных запросов, включая handleRequest, handlePersonalProfile, handleBuyAndRent, handleScanDocuments, handleMyDocuments и handleWorkflow. Каждый обработчик проверяет метод запроса и выполняет соответствующую логику в зависимости от типа запроса.
Установка и запуск
Для установки и запуска клиентской и серверной частей приложения, следуйте инструкциям ниже:
Клиентская часть
Установите Node.js и npm, если они еще не установлены на вашем компьютере.
Установите зависимости, выполнив команду npm install в корневой папке проекта.
Запустите клиентскую часть, выполнив команду npm start.
Серверная часть
Установите Go, если он еще не установлен на вашем компьютере.
Перейдите в папку с серверной частью проекта.
Запустите серверную часть, выполнив команду go run main.go.
Примечание: Убедитесь, что клиентская и серверная части выполняются на разных портах, чтобы избежать конфликтов.
